ğŸ§  How the Game Works

A random word is selected at the start.
Player guesses letters using the keyboard.
Correct guesses reveal letters in the word.

Wrong guesses:
Reduce coins
Draw parts of the hangman figure
The timer tracks how long the player takes.

The game ends when:
The word is guessed correctly (Win)
Maximum wrong guesses are reached (Loss)
Coins reach zero
